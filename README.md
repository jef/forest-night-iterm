# Forest Night - iTerm

![colortest](colortest.png)

## Install

- Download [`forest-night.itermcolors`](https://raw.githubusercontent.com/jef/forest-night-iterm/master/forest-night.itermcolors) and save it.
- Open iTerm and go to **Preferences** (<kbd>Cmd</kbd>+<kbd>,</kbd>) > **Colors** > **Color Presets...** > **Import...**
- Select `forest-night.itermcolors`.
